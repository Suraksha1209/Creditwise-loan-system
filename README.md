# Creditwise-loan-system

📊 Credit Wise – Loan Approval Prediction System
📌 Project Overview

Credit Wise is a Machine Learning project that predicts whether a loan application will be approved or not based on applicant details such as income, credit score, employment status, and more.

The goal of this project is to assist financial institutions in making data-driven, faster, and more accurate loan approval decisions.

🚀 Features
Data Cleaning & Preprocessing
Handling Missing Values
Exploratory Data Analysis (EDA) with visualizations
Feature Encoding (Label Encoding & One-Hot Encoding)
Feature Scaling using StandardScaler
Multiple ML Models:
Logistic Regression
K-Nearest Neighbors (KNN)
Naive Bayes
Model Evaluation using:
Accuracy
Precision
Recall
F1 Score
Best model selection based on performance
